Frequently Asked Questions

Which AI agents does ZingIt support? +
ZingIt supports Claude Code (via the Claude Agent SDK), GitHub Copilot CLI, and OpenAI Codex. You can switch between agents in the settings.
Can I use ZingIt on any website? +
Yes! The bookmarklet lets you inject ZingIt into any webpage. It uses Shadow DOM to isolate styles and avoid conflicts with the host page.
How do I undo changes made by the AI? +
ZingIt creates git checkpoints before each AI modification. Open the History panel to see all changes and click "Undo" to revert any modification.
Is ZingIt free to use? +
Yes, ZingIt is completely free and open source under the MIT license. You'll need your own API keys or CLI tools for the AI agents.
