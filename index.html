Streamline how you share UI changes with your AI assistant

Select → Annotate → Send to AI

Visual Annotations

Click any element on your page to add notes about changes you want. Smart CSS selectors are automatically generated to target the exact element.

Multi-Agent Support

Works with Claude Code, GitHub Copilot CLI, and OpenAI Codex. Choose your preferred AI assistant and watch it work in real-time.

Screenshot Capture

Automatically captures screenshots of annotated elements to provide visual context to AI agents. No more manual screenshot taking! ZingIt generates them for you with perfect element targeting.

Try ZingIt

1

Install an AI Agent

You'll need one of these installed:

  • Claude Code
  • GitHub Copilot CLI
  • OpenAI Codex
2

Start the Server

PROJECT_DIR=/path/to/your/project npx @codewithdan/zingit

Replace /path/to/your/project with your project path. Server runs on ws://localhost:3000

3

Ready to Try It?

Experience ZingIt in action with our interactive demo playground. Annotate forms, tables, buttons, and more with real-time feedback.

Launch Interactive Demo
4

Start Annotating

Press Z to toggle annotation mode, then click any element to annotate it.

Tip: Add ?zingit to any URL to auto-activate

5

Add to Your Page (Optional)

<script src="https://cdn.jsdelivr.net/npm/@codewithdan/zingit@latest/dist/zingit-client.js"></script>
<script>ZingIt.connect('ws://localhost:3000');</script>

For use on your own pages
